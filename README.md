# ハンドジェスチャの検証

MediaPipe Hand Landmarkerを使用して、ハンドジェスチャで画面上のオブジェクトを操作するデモアプリケーション。

# 使用技術

- **MediaPipe Tasks Vision** - 手の検出とランドマーク認識
- **TypeScript** - 型安全な開発
- **Vite** - 高速なビルドツール
- **Canvas API** - 描画とインタラクション

# デモ動画
https://youtu.be/oSOBgHdsDEE

# 機能

- リアルタイムの手の検出と骨格表示
- ピンチジェスチャーでオブジェクトをドラッグ&ドロップ
- 手が閉じた状態の検出（線の色が変化）
- スコアリングシステム（左側の四角形を右側に移動すると25点獲得）
- カメラ映像の左右反転（鏡のような操作性）

# 実行方法

## 環境

- Node.js 18以上
- Webカメラ
- モダンブラウザ（Chrome, Edge, Safari等）

## 動作手順

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173` を開く

## 操作方法

1. カメラに手を映す
2. 親指と人差し指を近づけてピンチ（40px以内）
3. ピンチしたまま手を動かすと四角形がドラッグされる
4. 指を離すとドロップ
5. 左側の四角形を右半分に移動すると25点獲得
